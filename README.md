### Installation and Setup
Requires `node v8.9.1` and `npm 5.5.1`.

```sh
$ git clone https://github.com/wmik/foodie.git
$ cd foodie
$ npm install
```

To run the app locally

`$ npm start`

and visit `localhost:3000`
